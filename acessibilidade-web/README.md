# Acessibilidade web ♿️

Projeto desenvolvido nos seguintes cursos:
- [Acessibilidade web: crie designs inclusivos](https://cursos.alura.com.br/course/acessibilidade-web-design-inclusivos)
- [Acessibilidade web parte 1: tornando seu front-end inclusivo](https://cursos.alura.com.br/course/acessibilidade-web-front-end)
- [Acessibilidade web parte 2: componentes acessíveis com um pouco de JavaScript](https://cursos.alura.com.br/course/acessibilidade-web-front-end-parte-2)

Todos da [Alura](https://alura.com.br/)

No projeto, pude ajustar elementos do HTML, CSS e JavaScript pensando em melhorar a acessibilidade do site, principalmente para pessoas com dificuldades de visão. Pude colocar em prática conceitos como modais acessíveis, testes com o NVDA, carrossel acessível, entre outros.
